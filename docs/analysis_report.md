## 2. 主要な知見

### 2.1 モデルアーキテクチャに関する知見

**Logistic回帰 > LightGBM**
- 同一特徴量セットにおいて、Logistic回帰がLightGBMを一貫して上回る。
- TeamRating(iter): Logistic 0.1703 vs LightGBM 0.1726 (差: 0.0023)
- TeamRating(both): Logistic 0.1709 vs LightGBM 0.1762 (差: 0.0053)
- 原因: トーナメント予測では「チーム力差→勝率」の関係がほぼシグモイド関数で表現できるため、線形モデルで十分。LightGBMは n_estimators=5 と小さく設定しているが、それでも過学習傾向がある。

**強い正則化が有効**
- キャリブレーション無しの場合: C=0.01 がベスト。
- Isotonic キャリブレーション付きの場合: C=0.05 付近がベスト（キャリブレーションが正則化の強さの違いを吸収するため、C値への感度が低下）。
- Ridge: α=100（強い正則化）がベスト。α=0.1は性能が低い。
- 48+特徴量に対してサンプル数が限られるため、多重共線性の影響を抑える強い正則化が必要。

**Isotonic キャリブレーションが有効**
- LogisticRegression の出力確率に Isotonic regression を適用することで Brier Score が改善: 0.1670 → 0.1663。
- 特に Men 側で効果が大きい（0.1939 → 0.1925）。
- Platt scaling (sigmoid) は LogisticRegression に対しては悪化する（シグモイドの二重適用）。
- `CalibratedModelWrapper` により、任意の ModelWrapper に汎用的にキャリブレーションを適用可能。

### 2.2 特徴量に関する知見

**男子（Men）の重要特徴量 Top 5**
1. `diff_elo` (0.171) — Eloレーティング差
2. `diff_iter_net` (0.139) — 反復調整法ネットレーティング差
3. `diff_sos` (0.132) — Strength of Schedule差
4. `diff_massey_top_std` (-0.121) — Massey上位システム標準偏差（負の寄与）
5. `diff_iter_off` (0.121) — 反復調整法オフェンスレーティング差

**女子（Women）の重要特徴量 Top 5**
1. `diff_sos` (0.335) — Strength of Schedule差
2. `diff_seed_num` (-0.334) — シード番号差
3. `diff_elo` (0.313) — Eloレーティング差
4. `diff_conf_elo_mean` (0.222) — カンファレンスElo平均差
5. `diff_conf_elo_median` (0.200) — カンファレンスElo中央値差

**男女間の特徴量重要度の違い**
- 女子はシード番号（公式評価）とSOS（対戦相手強度）の影響が非常に大きい → トーナメント結果がシード通りになりやすい
- 男子はEloと反復調整法レーティングが重要 → 番狂わせが多く、実力指標の精緻さが求められる
- 女子ではMassey Ordinals特徴量の係数がすべて0.0 → 女子にはMasseyデータが存在しないため、統合モデルでは自然にゼロ化

**Massey特徴量の貢献**
- Massey Ordinals（男子のみ）の追加により、男女別モデル → 統合モデルで Men Brier が改善: 0.1986 → 0.1939
- `diff_massey_top_std`（上位システム間の意見の不一致度）が負の係数 → 評価が割れるチームは予測困難であることをモデルが捉えている

### 2.3 統合モデル vs 男女別モデル

| 比較 | Combined | Separate (M+Wの加重平均) | 差 |
|------|:--------:|:------------------------:|:-:|
| Massey + Logistic C=0.01 | **0.1670** | 0.1695 | -0.0025 |
| Massey + Ridge α=100 | **0.1689** | 0.1698 | -0.0009 |

- 統合モデルが一貫して男女別モデルを上回る。
- 理由: 「チーム力差→勝率」の基本構造は男女共通であり、学習データ量の増加が正則化効果と相まってプラスに働く。
- ただし差は小さく（0.001〜0.003）、将来的に男女間の構造差が大きい特徴量を追加する場合は再検討が必要。

### 2.4 男女間の予測難易度の差

- Women Brier: 0.139〜0.145 → 予測しやすい（番狂わせが少ない）
- Men Brier: 0.194〜0.210 → 予測が難しい（番狂わせが多い）
- **Men の改善がスコア改善の最大の鍵**。Women側はすでにかなり良いスコアに達しており、改善余地は小さい。

### 2.5 Season間のばらつき

- 2023年が最もスコアが悪い (0.1879) → 番狂わせが多い年だった可能性
- 2025年が最もスコアが良い (0.1315) → 順当な結果が多かった可能性
- この年ごとのばらつきは本質的な予測困難性の反映であり、モデル改善で解消できる部分は限定的

## 3. 特徴量エンジニアリングの評価

### 現在使用している特徴量カテゴリ（51特徴量）

| カテゴリ | 特徴量数 | 貢献度 |

### コーチ特徴量の設計と効果

**背景**: 男子のみコーチデータ (MTeamCoaches.csv) が利用可能。全8特徴量を投入すると既存のElo/SeasonStats と高相関で冗長になり改善しない。

**相関分析に基づく特徴量選択**:
| コーチ特徴量 | Elo との相関 | 評価 |
|-------------|:----------:|------|
| coach_career_win_rate | 0.637 | ✗ 冗長 |
| coach_tourney_win_rate | 0.523 | ✗ 冗長 |
| coach_tourney_apps | 0.513 | ✗ 冗長 |
| coach_season_exp | 0.313 | △ やや冗長 |
| coach_tenure | 0.206 | ○ ユニーク |
| **coach_is_new** | **-0.119** | **◎ 最もユニーク** |
| **coach_mid_season_change** | **-0.042** | **◎ 最もユニーク** |

**結果**: minimal モード（tenure, is_new, mid_season_change の3特徴量）が最も効果的。
- Men Brier: 0.1925 → 0.1910 (-0.0015)
- 「新コーチの就任」「シーズン中の監督交代」というチーム不安定性のシグナルは、Elo やレーティングでは捕捉できない固有情報。
- full モード（8特徴量）や tournament モード（6特徴量）は冗長性で改善せず。

### 特徴量の冗長性問題

- `diff_elo`, `diff_iter_net`, `diff_margin`, `diff_net_eff`, `diff_win_pct` など、チーム力を測る指標が複数存在し多重共線性が発生。
- 強い正則化 + Isotonic キャリブレーションによって多重共線性の悪影響を緩和。
- 新特徴量の追加時は既存特徴量との相関を確認し、低相関のもののみ投入するのが有効。

## 6. 現状の課題

1. **LightGBM の活用が不十分**: n_estimators=5 は過度に保守的。Optuna による適切なチューニングで改善の余地あり。
2. **女子の Massey/Coach データ不在**: 女子にはMasseyデータもCoachデータもなく、男子のみの恩恵。女子独自の外部データがあれば有効。
3. **Season数の制限**: CV で 2021-2025 の5シーズンのみを検証に使用。サンプル数が少なくスコアのばらつきが大きい。
4. **2026年予測の不確実性**: 最終提出は2026年トーナメントの予測であり、CVスコアと実際のスコアは乖離する可能性がある。

## 8. アンサンブル戦略の詳細

### 3モデルの構成

| モデル | 正則化 | 特徴量 | 男子重み | 女子重み |
|--------|--------|--------|:--------:|:--------:|
| L2 (ScaledLogistic C=0.05 + Isotonic) | L2 (Ridge) | 全51特徴量 | 1.0 | 0.2 |
| L1 (Logistic C=0.01 + Isotonic) | L1 (Lasso) | 全51特徴量(自動選択) | 0.0 | 0.5 |
| Elo (ScaledLogistic C=0.1 + Isotonic) | L2 | Seed+Elo(6特徴量) | 0.0 | 0.3 |

### 根拠

**男子にL2のみ**:
- 男子は豊富な特徴量（Massey, Coach）が有効で、L2正則化で全活用が最適
- Eloブレンドは男子で効果なし（elo_w=0.0〜0.2で差なし）→ フルモデルがElo情報を既に内包
- L1は男子で悪化（0.1910→0.1968）: 特徴量の切り捨てが情報損失

**女子にL1主力**:
- 女子にはMassey/Coach特徴量が存在せず、使える特徴量が限定的 → 過学習リスクが高い
- L1の自動特徴量選択が過学習を防ぎ、Women Brier 0.1402→0.1367 と大幅改善
- 特徴量重要度分析で、女子の多くの特徴量が |coef| < 0.02 → ノイズ源の除去が有効

**女子にElo補完**:
- Elo単体でも女子は良好（0.1382）→ シンプルなモデルの安定性が女子で有効
- L1との相関が低く、異なるエラーパターン → ブレンド効果が大きい

## 9. 結論

現在のベストモデルは **3モデルアンサンブル (L2+L1+Elo, 男女別重み)** で、Combined Brier Score **0.1640** を達成。

改善の内訳（ベースライン 0.1670 からの推移）:
- **Isotonic キャリブレーション** (-0.0007): 出力確率を非パラメトリックに補正
- **Coach(minimal) 特徴量** (-0.0006): コーチの在任年数・新任・シーズン中交代
- **2モデルアンサンブル** (-0.0013): L2+Elo、女子でEloの安定性を活用
- **3モデルアンサンブル** (-0.0004): L1追加で女子の過学習をさらに抑制

総合改善: **0.1670 → 0.1640 (-0.0030, -1.8%)**

残りの最大課題は男子の0.1910。新特徴量の発見が鍵となる。
